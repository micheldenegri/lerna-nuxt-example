# @micheldenegri/nuxt

> A Nuxt.js app

## Development

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:9010
$ yarn dev

# lint
$ yarn lint

# run tests
$ yarn test

# build for production and launch server
$ yarn build
$ yarn start
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
